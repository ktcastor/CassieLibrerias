#Librerias

linuxapps.py
ES una libreria para poder ver que apps tenemos instaladas en nuestras distros de linux

Tienes que instanciar la clase LinuxApps()

solo tiene dos metodos:

  .currentApps()
     Te devuelve una lista con diccionarios que contienen el nombre de las apps instaladas y sus respectivos .desktop de la aplicación

  .getInfoApps()
    Te devuelve una lista con diccionarios que contienen información de las apps como sus nombres, icono y como ejecutarla en terminal
